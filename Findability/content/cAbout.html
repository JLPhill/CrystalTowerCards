Improving Findability

404 Error Page added

Included an error 404 webpage in order to improve the user's web experience when a erroneous page addresses are entered into the browser. This helps alleviate the issue while also giving the website an opportunity to redirect the user to a more suitable webpage within the site.

Meta Tags added

These were added to the php files of the page to include keywords that will improve the website's findability in search engines and provide more information for search engine crawlers to identify when indexing the page.

Https added

Made use of a SSL Certificate to change the website's HTTP to HTTPS. This change allows for a more secure website and improves searchability on google and other search engines that prioritize HTTPS webpages.

Keywords/Keyphrases added

Researched effective keywords and created phrases from those words. Added keywords to content and key phrases to improve findability and increase relevance in user searches.

Skip navigation added

Added skip navigation shortcuts to each webpage for more useability and better linkage between different parts of the website.

Converted all pages to PHP

Converted html pages to php in order to create a more efficient website and improve security.

Table added

Tables for displaying information and alt text.

.Htaccess file added

Improves findability by redirecting users to error page if there are issues with the web address and includes other useful functions that improve overall user experience.

Added meaningful alt text

Alt text is especially useful for web readers and visually impaired users. This helps increase useability and provides more information regarding images for search engine crawlers to index which improves findability.

Optimized images

Lowering the size of images and creating efficient visuals reduces the loading speed of a webpage.

Created readable/visual content

Using visually pleasing content with images and other forms of displayed information helps to improve SEO and readability.
